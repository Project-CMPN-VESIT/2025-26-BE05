# 💳 Credit Risk Assessment System

A machine learning-powered web application that evaluates the **creditworthiness of loan applicants** by predicting a **Credit Risk Score (300–900)** and **Default Probability**.

---

## 🚀 Overview

This project simulates a real-world **credit scoring system** used by financial institutions to assess the risk of lending money to individuals.

It combines:

* 📊 Machine Learning models
* 🌐 Interactive web interface
* 📈 Risk interpretation and explainability

---

## 🧠 Key Features

* 🔢 **Credit Risk Score Prediction** (300–900 scale)
* ⚠️ **Default Probability Estimation**
* 📉 **Risk Categorization**
* 🧾 **User-friendly Input Form**
* 🔁 **Auto-fill from Previous User Data**
* 📊 **Explainability (LIME / SHAP)**
* 🌙 **Dark Mode UI Support**

---


###  Credit Risk Score

* Range: **300–900**
* Higher score = Lower risk
  
---

## 🔍 Explainability

The system uses:

* SHAP (SHapley Additive exPlanations)

To:

* Identify important features
* Explain individual predictions

---

## 📌 Tech Stack

* Python
